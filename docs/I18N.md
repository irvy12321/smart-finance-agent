# Internationalization (i18n) Guide

This guide explains how to use the internationalization system in Smart Finance Agent.

## Overview

The frontend uses `react-i18next` for internationalization with support for:
- English (en)
- Simplified Chinese (zh-CN)

## Setup

The i18n system is automatically initialized in `src/main.tsx`. Language preference is saved in `localStorage`.

## Usage

### In Components

```tsx
import { useTranslation } from 'react-i18next'

function MyComponent() {
  const { t } = useTranslation()
  
  return (
    <div>
      <h1>{t('dashboard.title')}</h1>
      <p>{t('dashboard.welcome')}</p>
    </div>
  )
}
```

### With Interpolation

```tsx
// Language file
{
  "greeting": "Hello, {{name}}!"
}

// Component
const { t } = useTranslation()
t('greeting', { name: 'John' }) // "Hello, John!"
```

### With Pluralization

```tsx
// Language file
{
  "items": "{{count}} item",
  "items_other": "{{count}} items"
}

// Component
t('items', { count: 5 }) // "5 items"
```

## File Structure

```
src/i18n/
├── index.ts           # i18n configuration
├── i18next.d.ts       # TypeScript type definitions
└── locales/
    ├── en.json        # English translations
    └── zh-CN.json     # Chinese translations
```

## Adding a New Language

1. Create a new translation file in `src/i18n/locales/` (e.g., `ja.json`)
2. Import and add it to `src/i18n/index.ts`:

```typescript
import ja from './locales/ja.json'

i18n.use(initReactI18next).init({
  resources: {
    en: { translation: en },
    'zh-CN': { translation: zhCN },
    ja: { translation: ja },  // Add this
  },
  // ...
})
```

3. Add the language to `LanguageSwitcher`:

```typescript
const languages = [
  { code: 'en', name: 'English', flag: '🇺🇸' },
  { code: 'zh-CN', name: '简体中文', flag: '🇨🇳' },
  { code: 'ja', name: '日本語', flag: '🇯🇵' },  // Add this
]
```

## TypeScript Support

The `i18next.d.ts` file provides type safety for translation keys. When you add new keys to the English translation file, TypeScript will automatically validate them.

## Custom Hook

Use the `useAppTranslation` hook for additional convenience:

```tsx
import useAppTranslation from '../hooks/useAppTranslation'

function MyComponent() {
  const { t, currentLanguage, isChinese, isEnglish, changeLanguage } = useAppTranslation()
  
  if (isChinese) {
    // Chinese-specific logic
  }
  
  return <div>{t('key')}</div>
}
```
